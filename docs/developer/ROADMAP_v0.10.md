# Historical v0.10 planning document

> **Status: archived.** This document is retained only as historical context for the early publication-companion roadmap. It is not an active implementation plan and must not be used to determine current or future project work.

The authoritative development roadmap is [`docs/ROADMAP.md`](../ROADMAP.md). Its installed package copy is `inst/doc/ROADMAP.md`; those two files are maintained as byte-identical representations of the current project state.

Implementation history is recorded in merged pull requests, `NEWS.md`, and the open GitHub tracking issues referenced by the authoritative roadmap.

This archived document previously described Phase 7.3 manuscript generation, workflow execution, checkpoint/resume, scheduling, publication, and related capabilities as future work. Those descriptions became obsolete after the corresponding implementation phases were completed and are intentionally not reproduced here, to avoid creating a competing roadmap.
